# iOS Native App

This directory contains a SwiftUI implementation of the Neuroshima Hex native mobile app.

## What It Uses

- Bundled shared content from `NeuroshimaHexNative/Resources/Generated/`
- Bundled copied images from `NeuroshimaHexNative/Resources/Images/`
- Generated Apple localization catalog in `NeuroshimaHexNative/Resources/Localizable.xcstrings`

## Implemented Screens

- Army browser with search and feature switching
- Army detail
- Deck setup
- Draw mode
- Counter mode
- Tile flip mode

## Notes

- The project is provided as a native Xcode project plus shared scheme.
- Shared data and resource exports are generated by `../scripts/generate-native-shared.mjs`.
- `xcodebuild` could not complete in this environment because the local Xcode installation is missing a required simulator plug-in dependency. IDE lints did not report Swift errors in the new source file.
