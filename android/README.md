# Android Native App

This directory contains a Jetpack Compose implementation of the Neuroshima Hex native mobile app.

## What It Uses

- Bundled shared content from `app/src/main/assets/generated/`
- Bundled copied images from `app/src/main/assets/images/`
- Generated Android string resources in `app/src/main/res/values/` and `app/src/main/res/values-pl/`

## Implemented Screens

- Army browser with search and feature switching
- Army detail
- Deck setup
- Draw mode
- Counter mode
- Tile flip mode

## Notes

- The project is designed to be opened in Android Studio.
- This environment did not provide a working Gradle installation or wrapper, so the source was lint-checked but not compiled here.
- Shared data and resource exports are generated by `../scripts/generate-native-shared.mjs`.
